HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML attributes are always defined in the beginning element, never the closing element
  • Charset is the character set, it allows us to use certain characters and symbols on the page
  • Elements within the body of HTML are known as child elements, the body is known as the parent
  • Element refers to the entire element from opening to closing tags
  • Tag refers to what appears inside angle brackets
  • Alt text is important for screen readers and assistive tech

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • There are 3 ways to style a webpage 1. Inline CSS 2. Internal CSS style sheet 3. External CSS style sheet
  • Most common is external CSS style sheet
  • Separation of concerns refers to separating code to different files and having external access
  • Selector defines the element or attributes to which rules or declarations will apply
  • Declarations contain two important componenets, the CSS property and the value of the property
  • DRY- Do not repeat yourself. This tells developers to reuse and share pieces of code in order to reduce overall size and complexity of a code base
  • For colour you can use the name of the colour or the hex number of that colour

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name:creates a new brnach and switches to it
  • git checkout main switches back to the main branch
  • MIT licence md which stands for markdown, light weight plain text formatting language
  • Blocker- a particular problem that you are stuck on
  • pwd displays current working directory
  • ls displays the list of files and folders in current directory
  • cd navigates between current directories
  • cd . navigates backwards one level at a time
  • mkdir creates a new directory
  • touch creates a file inside a directory
  • start opens a files
